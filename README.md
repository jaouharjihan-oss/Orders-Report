# Orders-Report
📊 Tableau Dashboard: Orders Report Analysis
🎯 Project Overview
This project features an interactive dashboard created with Tableau to analyze business order performance. The primary goal is to transform raw sales data into actionable insights by tracking key performance indicators (KPIs) and identifying growth opportunities.

🔗 https://public.tableau.com/app/profile/jihan.jaouhar/viz/Classeur1_17746532284100/OrdersReport?publish=yes

💡 Key Business Questions
This analysis provides data-driven answers to the following questions:

Sales Trends: What is the monthly/yearly evolution of order volume?

Product Performance: Which categories and sub-categories drive the most revenue?

Profitability: How does profit margin vary across different customer segments or regions?

Operational Efficiency: What are the average shipping delays per region?

🛠️ Technical Skills Demonstrated
Data Preparation: Connecting and cleaning datasets for accurate analysis.

Advanced Calculations: Creating Calculated Fields to derive custom KPIs and growth metrics.

Data Visualization: Designing intuitive charts (Time Series, Maps, Treemaps) to simplify complex data.

Interactivity: Implementing Dynamic Filters and Dashboard Actions for a seamless user experience.
